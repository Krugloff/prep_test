# Q5

## title

Which option can be inserted into `__(1)__`?

## body

**Assume that the following code must have the stated execution result:**

```
def round(n, __(1)__)
  n.round(__(1)__)
end

p round(2.5, half: :even)

[Execution Result]
2
```

## options

### A

`*`

### B

`&`

### C

`..`

### D+

`...`

## comment

`...` forwards rest arguments including keyword arguments and the block.
